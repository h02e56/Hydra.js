Hermes ( Level oLevel, Number nImmediate, Number nTimeout ) Private
Hermes is the class that manages Appender, Messages, Deferred calls and Levels of Messages to show We can add Message but if the Message is not of the same level the message will not be showed. If the Hermes Message Level is ALL all the logs will be added If the Hermes Message Level is OFF no log will be added The add of logs can be deferred to avoid multiple calls that block browser code »
